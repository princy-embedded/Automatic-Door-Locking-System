# Automatic Door Locking System

## Description

This project is an embedded-based automatic door locking system using Arduino and sensor interfacing.

The system automatically locks or unlocks the door based on sensor detection and control logic.

## Features

- Automatic door locking
- Sensor-based detection
- Real-time control
- Embedded automation

## Components Used

- Arduino UNO
- Servo Motor
- Ultrasonic Sensor / PIR Sensor
- Jumper Wires
- Breadboard

## Technologies Used

- Embedded C
- Arduino IDE

## Working Principle

The sensor detects nearby movement or object presence. Arduino processes the sensor input and controls the servo motor to lock or unlock the door automatically.

## Applications

- Smart homes
- Security systems
- Office automation
- IoT-based automation
